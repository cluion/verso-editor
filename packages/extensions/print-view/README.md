# @verso-editor/extension-print-view

Print View extension for Verso Editor. Adds `@media print` CSS styles and a `print()` command.

## Install

```bash
pnpm add @verso-editor/extension-print-view
```

## Usage

```typescript
import { PrintViewExtension } from '@verso-editor/extension-print-view'
```

Included in `createStarterKit()`.
