# @verso-editor/extension-rtl

RTL (right-to-left) text direction extension for Verso Editor.

## Features

- Toggles `dir` attribute (`ltr`/`rtl`) on paragraph and heading nodes
- Keyboard shortcut: `Mod-Alt-d`

## Install

```bash
pnpm add @verso-editor/extension-rtl
```

## Usage

```typescript
import { RtlExtension } from '@verso-editor/extension-rtl'
```

Included in `createStarterKit()`.
