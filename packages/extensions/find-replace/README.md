# @verso-editor/extension-find-replace

Find & Replace extension for Verso Editor. Highlights search results with decorations.

## Features

- Case-sensitive and regex search options
- `find()`, `replace()`, and `replaceAll()` commands
- Configurable `onResults` callback for UI integration

## Install

```bash
pnpm add @verso-editor/extension-find-replace
```

## Usage

```typescript
import { FindReplaceExtension } from '@verso-editor/extension-find-replace'
```

Included in `createStarterKit()`.
